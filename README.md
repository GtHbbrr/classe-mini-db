# Classe Mini 6.50 – Open Database ⚓

The most complete, structured and up-to-date historical database of the Classe Mini 6.50 – **free forever**.

- 1,400+ boats (with Scow / Maxi / Foiling / Pointy typing)  
- 15,000+ race results (1977–2025)  
- 800+ skippers (gender, nationality, training base, pedigree)  
- Ownership & sales history (with "disponible" pitches)  
- Weekly automatic updates  

Live dashboard → https://mini.db.fyi (coming soon)  
Download full SQLite → /data/exports/classe_mini_historical.db  

Made with love for the craziest 6.50 sailors on the planet.

## Quick Start
1. Clone: `git clone https://github.com/GtHbbrr/classe-mini-db.git`
2. Run: `python src/build_database.py`
3. Explore: Open `data/exports/classe_mini_2025_full.xlsx`
